# 🤖 TrAI-Bot : AI Server (Python)

FastAPI와 Python으로 구현된 **TrAI-Bot** 프로젝트의 AI 서버 레포지토리입니다.

LLM 추론, RAG(검색 증강 생성) 파이프라인, 임베딩 처리 등 AI 관련 핵심 기능을 전담하여 처리합니다.
